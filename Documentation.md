1. Introduction
The growth of e-commerce has significantly changed the way consumers purchase products and services. Customers increasingly expect online shopping platforms to provide smooth navigation, intuitive interfaces, responsive layouts, and secure transactions.
This project presents the design and prototype of an Online Shopping Web Application developed using Figma. The system was designed following modern UI/UX principles to create an engaging, accessible, and user-friendly shopping experience. The application allows users to browse products, manage shopping carts, save favorite items, and complete purchases efficiently.
The project demonstrates practical application of user interface design, user experience design, wireframing, prototyping, accessibility principles, and responsive design techniques.
________________________________________
2. Problem Statement
Many online shopping systems face usability challenges that negatively affect customer satisfaction and purchasing behavior. Common issues include:
•	Complex navigation structures.
•	Poor product organization.
•	Difficult product search experiences.
•	Lengthy checkout processes.
•	Inconsistent user interfaces.
•	Lack of mobile responsiveness.
These challenges often result in abandoned shopping carts and reduced user engagement.
The Online Shopping Web Application aims to solve these problems by providing a clean, intuitive, and efficient shopping platform that enables users to quickly discover products, manage purchases, and complete transactions with minimal effort.
